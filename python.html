Python

Python on interpreteeritav, dünaamiliselt tüübistatav, objektorienteeritud ja kõrgetasemeline programmeerimiskeel. Python on tuntud kui algajatele sobiv keel, mis on asendanud Java kui kõige levinuma algõppekeele, kuna see võtab enda kanda suure osa keerukusest, võimaldades algajatel keskenduda programmeerimise põhimõtete omandamisele, mitte pisidetailidele.

Põhiomadused

  • Dünaamiline tüübistamine – muutujate tüüpi ei pea ette määrama.
  • Mitme paradigma tugi – objektorienteeritud, funktsionaalne ja protseduuriline.
  • Platvormiülene – töötab Windowsil, Linuxil, macOS-il jpm.
  • Interpretaatoripõhine – ei vaja kompileerimist.

Kasutusvaldkonnad

  • Veebiarendus: Django, Flask.
  • Andmeteadus: NumPy, Pandas, Matplotlib.
  • Masinõpe: TensorFlow, PyTorch, scikit-learn.
  • GUI rakendused: Tkinter, PyQt.
  • Mängude arendus: Pygame.

Süntaksi näide

# Funktsioon, mis võtab nimekirja numbreid ja tagastab uue nimekirja,
# kus iga number on suurendatud 1 võrra
def suurenda(numbrid):
    # list comprehension – lühike viis uue listi loomiseks
    return [n + 1 for n in numbrid]

# Funktsioon, mis leiab ainult paarisarvud
def paaris(numbrid):
    tulemus = []
    for n in numbrid:
        if n % 2 == 0:  # % kontrollib jääki, paarisarv on jagatav kahega
            tulemus.append(n)
    return tulemus

# Funktsioon, mis tagastab numbrite summa
def summa(numbrid):
    s = 0
    for n in numbrid:
        s += n
    return s

# Funktsioon, mis prindib eri käitumise sõltuvalt väärtusest
def kirjeldus(n):
    if n < 0:
        print("Negatiivne arv:", n)
    elif n == 0:
        print("Null")
    else:
        print("Positiivne arv:", n)

# Testime funktsioone
numbrid = [1, 2, 3, 4, 5]
print("Algne list:", numbrid)
print("Suurendatud:", suurenda(numbrid))
print("Paarisarvud:", paaris(numbrid))
print("Summa:", summa(numbrid))

kirjeldus(-2)
kirjeldus(0)
kirjeldus(8)
