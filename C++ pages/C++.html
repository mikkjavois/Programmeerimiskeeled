C++

Kiir ja võimas programmeerimiskeel, mida kasutatakse keerukate ja tõhusate programmide loomiskes. See keel sobib nii tavaliste lihtsate programmide kui ka projektide jaoks, mis on üles ehitatud igasugustest objektidest ja osadest. C++ võimaldab programmeerijal täpselt hallata mälu ja arvuti tööd, mis aitab programmidele kiiremini töötama. Tänu sellele kasutatakse seda keelt peaaegu kõikjal, mis teeb seda keelt universaalseks!

Põhiomadused

Objektorienteeritud programmeerimine

Võimaldab teha “objekte”, mis on nagu väiksed programmid programmis

Madala taseme juhtimine

Saad ise kontrollida, kuidas arvuti mälu kasutatakse.

Tugev tüübisüsteem

Arvuti kontrollib juba enne programmi käivitamist, et kõik töötas korralikult.

Standardteek

C++ omab suure kogumi tööriistu ja andmestruktuuri, mida saab kohe kasutama.

Kasutusvaldkonnad

  • Operatsioonisüsteemid -> Windows, macOS (osaliselt)
  • Seadme draiverid -> NVIDIA / AMD graafikadraiverid
  • Mikrokontrollerid -> Arduino, STM32
  • Kauplemisplatvormid -> Bloomberg Terminal, MetaTrader
  • Mängumootorite arendus -> Unreal Engine, CryEngine
  • Arvutimängude loomine -> Fortnie, World of Warcraft

Koodi näidis


                // TEKSTI VÄLJAKIRJUTAMINE \\
                #include <iostream>         //Alustame sisend ja väljundi faili sisaldamisega//
                int main() {                      //Siis tuleb kirjutama funktsiooni//
                    std::cout << "Tere maailm!";  //Viimasena võtame raamatukogust (std) teksti väljastamine (cout) ja kirjutame!//
                }

                // ARVUTAMINE \\
                #include <iostream>
                using namespace std;              //Et koguaeg mitte välja kirjutama "std::" //
                int main() {
                    int a;                        //Kirjutame välja, millega meie arvud hakkavad võrduma//
                    int b;                        //Sõna "int" on muuta lihtsalt arvud täisarvudeks//

                    cout << "Sisendage arvud:" ;  //Nüüd teeme nii, et meie sisendatud arvud olid võrdset kahe meie muutujaga//
                    cin >> a;
                    cin >> b;

                    int sum = a + b;    

                    cout << "Summa:" << sum;      //Viimasena väljastame summat//
                }

                // If/Else LAUSE \\
                #include <iostream>
                using namespace std;
                
                int main() {
                    int number;
                    cout << "Sisestage numbrid: ";
                    cin >> number;
                    
                    if (number % 2 == 0) {     //Alustame if lausega ja vaatame, kas arv on paaris arv//
                        cout << "Paaris arv";
                    } else {                   //Kui arv ei ole paaris, siis ta on paaritu//
                        cout << "Paaritu arv";
                    }
                }

                // TSÜKKEL \\
                #include <iostream>
                using namespace std;
                
                int main() {
                    int i = 1;      //Teeme nii, et muutuja i võrdub 1//
                    
                    for (;i <= 10; i+=1) {  //Kirjutame tsükkli, kus sulgude alguses on ";", kuna i on juba määratud.    
                        cout << i << " ";   Pärast kirjutame tsükkli lõpetamise tingimust ja lõpuks kui palju kasvab muutuja i.//
                    }
                }
