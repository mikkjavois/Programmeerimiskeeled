JavaScript

JavaScript on paindlik ja laialdaselt kasutatav programmeerimiskeel, mis kujundab suure osa tänapäevasest veebist. Algselt loodud veebilehtedele interaktiivsuse lisamiseks, nüüdseks arenenud täisväärtuslikuks programmeerimiskeeleks, mida kasutatakse nii brauseris kui ka serveris, mobiili- ja töölauarakendustes ning isegi nutiseadmetes. Tänu oma universaalsusele ja rohketele teekidele on JavaScriptist saanud üks kaasaegse arenduse põhikeeli.

Põhiomadused

Dünaamiline Tüüpimine

Muutuja väärtuse tüüp selgub käitusajal, võimaldades paindlikku ja kiiret arendamist.

Lihtne Objektipärimine

Keele pärimismudel põhineb objektidel, mis saavad üksteist laiendada ja täiendada.

Asünkroonne Käitumine

Sündmuspõhine mudel koos callback'ide, promise'ide ja async/await toega võimaldab tõhusalt hallata viivitusega toiminguid.

Lai Kasutusala

JavaScript töötab brauserites, serverirakendustes, mobiili- ja lauaarvutirakendustes ning isegi IoT seadmetes.

Kasutusvaldkonnad

  • Veebirakendused: Interaktiivsed kasutajaliidesed, reaalajas toimivad rakendused nagu vestlus- või kaardirakendused.
  • Serveripoolne arendus: Node.js võimaldab luua kiireid, skaleeritavaid ja sündmuspõhiseid servereid, mis sobivad nii API-de kui ka suure koormusega teenuste jaoks.
  • Mobiilirakendused: Raamistikud nagu React Native, Ionic ja NativeScript võimaldavad mugavalt ehitada mobiilirakendusi.
  • Töölauarakendused: Electroni ja Tauri abil saab luua platvormidevahelisi töölauarakendusi; tuntud näited on VS Code, Discord, Slack ja Figma.
  • Mänguarendus: JavaScript sobib 2D- ja 3D-veebimängude loomiseks tänu teekidele nagu Phaser, Three.js ja Pixi.js, pakkudes graafilisi efekte ja animatsioone brauseris.
  • Masinõpe: TensorFlow.js ja Brain.js võimaldavad käivitada närvivõrke ja treenida masinõppe mudeleid otse brauseris või Node.js-is, ilma serveripoolse töötluseta.

Süntaksi näide

// Objekt kasutaja andmetega
const kasutaja = {
    nimi: "Kati",
    vanus: 28,
    hobid: ["joonistamine", "matkamine"],

    // Meetod (funktsioon), mis tagastab lühikese kirjelduse
    kirjeldus() {
        return `${this.nimi} on ${this.vanus}-aastane ja talle meeldib ${this.hobid[0]}.`;
    }
};

console.log(kasutaja.kirjeldus());

// Klassi näide
class Ulesanne {
    constructor(pealkiri) {
        this.pealkiri = pealkiri;
        this.tehtud = false;
    }

    teeValmis() {
        this.tehtud = true;
    }
}

const u1 = new Ulesanne("Õpi JavaScripti");
u1.teeValmis();
console.log(u1);

// Lihtne asünkroonne funktsioon
async function laeAndmed() {
    // Simuleerime andmete laadimist Promise abil
    const tulemus = await new Promise(resolve =>
        setTimeout(() => resolve("Andmed laetud!"), 500)
    );
    console.log(tulemus);
}

// Käivita asünkroonne funktsioon ja jätka
laeAndmed();

// Või oota asünkroonse funktsiooni tulemus enne jätkamist
await laeAndmed();
