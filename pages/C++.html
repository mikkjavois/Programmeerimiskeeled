C++

Kiir ja võimas programmeerimiskeel, mida kasutatakse keerukate ja tõhusate programmide loomiskes. See keel sobib nii tavaliste lihtsate programmide kui ka projektide jaoks, mis on üles ehitatud igasugustest objektidest ja osadest. C++ võimaldab programmeerijal täpselt hallata mälu ja arvuti tööd. mis aitab programmidele kiiremini töötama. Tänu sellele kasutatakse seda keelt peaaegu kõikjal, mis teeb seda keelt universaalseks!

Kasutused: 1. Operatsioonisüsteemid -> Windows, macOS (osaliselt) 2. Seadme draiverid -> NVIDIA / AMD graafikadraiverid 3. Mikrokontrollerid -> Arduino, STM32 4. Kauplemisplatvormid -> Bloomberg Terminal, MetaTrader 5. Mängumootorite arendus -> Unreal Engine, CryEngine 6. Arvutimängude loomine -> Fortnie, World of Warcraft

Põhiomadused: 1. Objektorienteeritud programmeerimine Võimaldab teha “objekte”, mis on nagu väiksed programmid programmis 2. Madala taseme juhtimine saad ise kontrollida, kuidas arvuti mälu kasutatakse. 3. Tugev tüübisüsteem arvuti kontrollib juba enne programmi käivitamist, et kõik töötab korralikult. 4. Standardteek C++ omab suure kogumi tööriistu ja andmestruktuuri, mida saab kohe kasutama.
